# bitrader
Bitcoin Arbitrage utilities

## Quickstart
```
git clone https://github.com/jr-minnaar/bitrader.git
cd bitrader
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp example.env .env
```

Edit the .env file and add your Telegram bot url.

```
python arbot.py
```
